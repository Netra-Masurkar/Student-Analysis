Student Management Analysis
Project Overview

The Student Management Analysis project is a Python-based data analysis application that evaluates student academic performance using generated student data. The project performs data cleaning, analysis, visualization, ranking, grading, and performance evaluation to derive meaningful insights about students.

Features
.Generate student academic records using NumPy.
.Store and manage data using Pandas DataFrames.
.Analyze student marks across multiple subjects.
.Calculate total marks and percentage.
.Evaluate attendance records.
.Identify students scoring above a threshold.
.Generate student result status (Pass, Fail, ATKT).
.Assign grades based on percentage.
.Rank students according to performance.
.Visualize data using Matplotlib and Seaborn.
