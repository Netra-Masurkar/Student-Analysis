#  Student Management Analysis

## Project Overview

Student Management Analysis is a Python-based data analysis project that evaluates student academic performance using generated student data. The project uses Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn to perform data analysis and visualization.

The objective of this project is to analyze student marks, attendance, grades, and overall performance to extract meaningful insights.

---

##  Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

##  Dataset Information

The dataset contains the following information:

* Student ID
* Gender
* Standard (Class)
* Mathematics Marks
* English Marks
* Science Marks
* Social Studies Marks
* Computer Marks
* Attendance Percentage
* Previous Performance

##  Grading System

| Percentage       | Grade |
| ---------------- | ----- |
| 80% and Above    | A     |
| 70% - 79%        | B     |
| 50% - 69%        | C     |
| 35% - 49%        | D     |
| Below 35% / ATKT | F     |
